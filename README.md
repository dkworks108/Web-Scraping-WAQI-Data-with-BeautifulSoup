# WAQI AQI Scraper 

A simple Python web scraping project that extracts location data from the WAQI website using **BeautifulSoup** and stores it in a **Pandas DataFrame**.

##  Features
- Scrapes AQI-related location data
- Uses BeautifulSoup with lxml parser
- Converts scraped data into a Pandas DataFrame

##  Tech Stack
- Python
- Requests
- BeautifulSoup (bs4)
- Pandas
- lxml
